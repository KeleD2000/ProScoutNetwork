Admin felület

Admin felület mire jó?

Az admin felületet arra a célra van, hogy ha esetleg egy játékoskereső, úgy véli, hogy egy játékos hamis adatokat adott meg magáról, és nem fedi a valóságot, akkor tudja jelenteni, és az admin tudja kezelni ezeket az eseteket. Utána jár és ha úgy véli, hogy tényleg nem valós adatokat töltött fel magáról, akkor azt a felhasználót tudja törölni.
